Manual do Sistema de Gerenciamento

Navegação Principal

A barra de navegação no topo da tela permite acessar as principais seções do sistema:

  • Home: Página inicial com informações gerais e documentação do sistema.
  • Estoque: Gerenciamento de produtos em estoque.
  • Vendas: Registro e acompanhamento de vendas.
  • Fornecedores: Gerenciamento de informações de fornecedores.
  • Clientes: Gerenciamento de informações de clientes.
  • Contas: Controle de contas a pagar e a receber.

Funcionalidades Detalhadas

1. Estoque

  • Visualização de Produtos: Acesse a página "Estoque" para visualizar todos os produtos cadastrados.
  • Cadastro de Produtos: Clique em "Cadastrar Produto" para adicionar um novo produto.
  • Edição de Produtos: Na lista de produtos, clique em "Editar" ao lado do produto desejado.

2. Vendas

  • Registro de Vendas: Acesse a página "Vendas" e clique em "Nova Venda".
  • Visualização de Vendas: Acesse a página "Vendas" para visualizar o histórico de vendas.

3. Fornecedores

  • Visualização de Fornecedores: Acesse a página "Fornecedores" para visualizar todos os fornecedores.
  • Cadastro de Fornecedores: Clique em "Cadastrar Fornecedor" para adicionar um novo fornecedor.
  • Edição de Fornecedores: Na lista de fornecedores, clique em "Editar" ao lado do fornecedor desejado.

4. Clientes

  • Visualização de Clientes: Acesse a página "Clientes" para visualizar todos os clientes.
  • Cadastro de Clientes: Clique em "Cadastrar Cliente" para adicionar um novo cliente.
  • Edição de Clientes: Na lista de clientes, clique em "Editar" ao lado do cliente desejado.

5. Contas

  • Contas a Pagar: Acesse "Contas" -> "A Pagar" para visualizar as contas a pagar.
  • Contas a Receber: Acesse "Contas" -> "A receber" para visualizar as contas a receber.

Observações

  • Certifique-se de preencher todos os campos obrigatórios nos formulários.
  • Utilize a barra de navegação para alternar entre as seções do sistema.
  • Em caso de dúvidas, consulte a documentação disponível na página inicial.
